# Literary Studio — полный набор мокапов

Десктоп (01-14), мобильный (19-26), системные экраны (15-18).
Каждый .html открывается напрямую в браузере.

## Десктоп
01 — Написание (редактор со статусом, стоп-листом, AI-панелью)
02 — Планирование: Матрица романа
03 — Планирование: Таймлайн
04 — Планирование: Карта связей (с табом Карточка/AI)
05 — Планирование: Статистика (сводка + график темпа)
06 — Карточка персонажа с аркой
07 — Реквизиты книги (таб 1 из 3)
08 — Продакшн (таб 2 из 3)
09 — Публикация (таб 3 из 3)
10 — Форма сущности Библии мира (шаблон на примере Локации — переиспользуется для Организации/Правила/Термина)
11 — Авторский стиль (Стоп-лист с примерами)
12 — Версии и diff
13 — Контекстное меню действий (замена модалки для десктопа)
14 — Панель настроек

## Мобильный (Android)
19 — Библиотека / дерево навигации
20 — Редактор (статус-дропдаун + инлайн стоп-лист)
21 — Карточка персонажа
22 — Матрица романа в режиме просмотра (паттерн переиспользуется для Таймлайна/Карты связей/Статистики — широкие десктопные виды сворачиваются в вертикальный список карточек)
23 — Помощники (структурированные находки вместо чата)
24 — Шапка — два состояния (без книги / с книгой)
25 — Bottom sheet: меню действий
26 — Bottom sheet: настройки

## Системные экраны
15 — UI kit (кнопки, поля форм, бейджи)
16 — Вход / авторизация
17 — Тосты и модальное окно ошибки
18 — Индикатор сохранения

## Не отрисованы отдельно (переиспользуют паттерн уже нарисованного экрана)
- Мобильные версии Таймлайна/Карты связей/Статистики/Версий/Реквизитов/форм Библии мира/Авторского стиля — следуют тому же паттерну адаптации, что в 22 (десктопная таблица/граф → вертикальный список карточек, форма → одна колонка на весь экран)
- Регистрация — та же форма, что 16, плюс поле подтверждения пароля
